Link Files
==========

**Symbolic Link**
-----------------

**Directories**

::

	ln -s /base/* /target/ ** ls -l /target/

**Single Files:**

::

	ln -s Existing-file New-name

**Overwrite Existing Destination Files**

::

	sudo ln -sf Existing-file New-name
