File Locations
===============

+------------+--------------------------------+
| Location   | Linux                          |
+============+================================+
| $VIM       | ~/../../usr/share/vim/         |
+------------+--------------------------------+
| $VIMHOME   | ~/                             |
+------------+--------------------------------+
| $MYVIMRC   | ~/.vimrc                       |
+------------+--------------------------------+
| $VIMRUNTIME| ~/../../usr/share/vim/vim80/   |
+------------+--------------------------------+

+-----------+------------------------------------------------------------------------+
| Helpful Vim Commands -> Document References                                        |
+============================================+=======================================+
| :echo glob($VIMRUNTIME . '/syntax/\*.vim') | Lists Vim FileTypes                   |
+--------------------------------------------+---------------------------------------+
| :so $VIMRUNTIME/syntax/hitest.vim          |  shows complete list of color groups  |
+--------------------------------------------+---------------------------------------+
