Vim Tips
########
:Author: David Boyd
:Date: 2020-08-03

These commands are from Vim defaults unless otherwise stated.

Insert Mode
===========

.. code-block :: Vim

  " Jump down (next line)
  <C-o>j
  
  " Jump up (previous line)
  <C-o>o

Command Mode
============

.. code-block :: Vim

  " Basic statusline bar (case no plugins)
  set ruler
