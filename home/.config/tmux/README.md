### Step 1: Install Packages

1. Install TMUX (if not already installed): `sudo pacman -S tmux`
2. Install the [TMUX Package Manager (TPM)][tpm]:

``` bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
